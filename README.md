# Fliiq

The AI agent you actually own. One command to plan, build, and automate — with persistent memory and self-generating skills.

[Website](https://fliiq.ai) · [Documentation](https://docs.fliiq.ai) · [Changelog](./CHANGELOG.md)

## Install

```bash
pip install fliiq
fliiq init
fliiq
```

## Features

- Agent loop with 51 core skills
- Self-generating skills (detects gaps, builds + tests new skills automatically)
- Persistent memory across sessions
- Email, SMS, Telegram, Slack, Discord, WhatsApp, voice calls
- Google Workspace (Calendar, Gmail, Drive, Sheets, Docs, Slides)
- Browser automation
- Daemon with scheduled jobs (cron, interval, webhook)
- Colony multi-agent system
- MCP server support
- Full-screen TUI
